MediaConfigTool - Phase 1

Instructions

1. Extract all files from the ZIP to a local folder.
2. Run MediaConfigTool.exe
3. Click "Connect to Database"
4. Select a tenant
5. Click "Browse Folder"
6. Select a local image directory

Notes

- Internet connection is required to connect to Supabase.
- If no tenants appear, the database may be empty or unavailable.
- This delivery corresponds to Phase 1 of the desktop tool.
